# 智能复盘助手使用说明

## 简介

智能复盘助手是一款帮助用户进行日常复盘和未来规划的工具。它可以帮助您记录每日的目标达成情况、优点、不足和改进措施，并根据四象限时间管理法则进行明天的规划。同时，系统还会根据您的历史数据提供智能成长建议。

## 核心功能

### 1. 每日复盘

每天晚上，您可以记录以下信息：

- **目标达成情况**：设置并记录当天目标的完成百分比
- **优点**：记录当天做得好的地方
- **不足**：记录需要改进的地方
- **改进措施**：针对不足提出具体的改进方法
- **待办事项**：列出未完成的任务，用于明天规划

### 2. 明日规划（四象限法则）

根据时间管理四象限法则规划明天的任务：

- **A类：重要且紧急**：需优先处理的任务
- **B类：重要不紧急**：需要长期规划的任务
- **C类：紧急不重要**：可以委托或批量处理的任务
- **D类：不重要不紧急**：尽量减少或取消的任务

每个任务都包含：
- 做什么（任务内容）
- 为什么做（重要性）
- 怎么做（执行方法）
- 遇到问题如何解决

### 3. 历史记录管理

- 按月份查看历史复盘记录
- 查看详细的复盘和规划内容
- 删除不需要的历史记录

### 4. 智能建议

系统会根据您的历史复盘数据，提供个性化的成长建议，包括：
- 目标达成趋势分析
- 优势强化建议
- 改进方向指导
- 习惯养成建议
- 时间管理优化
- 成就回顾与激励

## 使用方法

### 1. 访问智能复盘助手

通过浏览器访问：`http://localhost:8080/review-assistant.html`

### 2. 进行每日复盘

1. 在"复盘"标签页中，填写当天的目标达成情况
2. 记录您的优点、不足和改进措施
3. 点击"保存复盘"按钮保存记录

### 3. 规划明天的任务

1. 在"规划"标签页中，根据四象限法则分类添加任务
2. 为每个任务填写完整的信息
3. 点击"保存规划"按钮保存记录

### 4. 查看历史记录

1. 切换到"历史"标签页
2. 使用月份筛选器查看特定月份的记录
3. 点击记录卡片查看详细内容

### 5. 获取智能建议

1. 切换到"建议"标签页
2. 系统会自动分析您的历史数据并生成建议
3. 点击"刷新建议"按钮可重新生成最新建议

## 数据存储

所有数据都保存在您的浏览器本地存储中，不会上传到任何服务器。您可以放心使用，数据只保存在您的设备上。

## 最佳实践

1. **坚持每日复盘**：养成每晚固定时间复盘的习惯
2. **具体量化**：尽量使用具体的数字和案例描述完成情况
3. **诚实面对不足**：勇于记录和分析自己的不足
4. **制定可执行计划**：改进措施和明日规划要具体、可操作
5. **定期回顾**：每周回顾一次历史记录，发现长期趋势

## 技术特点

- 纯前端实现，无需后端服务器
- 响应式设计，支持各种设备访问
- 现代化UI界面，流畅的用户体验
- 本地数据持久化存储
- 智能数据分析和建议生成

## 常见问题

**问**：我的数据保存在哪里？
**答**：所有数据都保存在您浏览器的本地存储中，仅在您的设备上可见。

**问**：如何查看之前的复盘记录？
**答**：切换到"历史"标签页，使用月份筛选器可以查看特定月份的记录。

**问**：建议是如何生成的？
**答**：系统会分析您的历史复盘数据，包括目标完成情况、优点、不足等，生成个性化的成长建议。

**问**：是否支持导出数据？
**答**：目前暂不支持数据导出功能，所有数据仅保存在本地存储中。
